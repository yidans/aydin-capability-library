# Figure and Table Evidence Audit

## Purpose

Use this card near the end of writing and review. The question is not only
whether figures and tables exist, but whether each visible exhibit supports the
exact claim made in the surrounding prose.

## Inventory First

Build or mentally check an inventory before judging quality:

- figure/table number or label
- source file or table path
- caption/title
- first in-text callout
- producing script or data source when available
- main-text versus appendix location

Every in-text callout must resolve to an existing exhibit. Every main exhibit
must be called out in prose before or near where it appears. Numbering should
be gap-free and non-duplicated.

## Evidence Consistency

For each substantive claim tied to a figure or table, match the claim to the
specific row, column, panel, model, or plotted quantity. Check direction,
magnitude, uncertainty, subgroup, denominator, and sample restriction. A claim
that says "higher," "lower," "significant," "robust," or "unchanged" must be
visible in the exhibit or traceable to the reported source table.

Do not infer exact values by eyeballing a plot unless labels provide the exact
numbers. If only the rendered image is available, mark the issue as requiring
author or source-data verification rather than inventing precision.

## Caption and Note Standards

- Figure captions name the plotted variables, units, groups, panels, uncertainty
  interval, and sample scope.
- Table titles name the estimand, comparison, or model family, not only
  "Results."
- Table notes define dependent variables, treatment coding, omitted categories,
  controls, fixed effects, clustering, weights, and significance markers when
  relevant.
- Abbreviations are defined on first use.
- Transformations, scale direction, exclusions, and subgroup restrictions are
  explicit.

## Production and Accessibility

Flag unreadable labels, cropped tables, unresolved LaTeX or Markdown references,
ambiguous legends, inconsistent decimal precision, missing units, color choices
that fail in grayscale, low-resolution raster text, and manual edits that are
not documented.

## Blocking Issues

Return for revision if any main figure/table is missing, contradicted by the
text, uncited, mislabeled, decorative, based on unexecuted data, clipped in the
rendered PDF, or described with a different design than it actually shows. A
treatment-level sweep is not a sensitivity analysis unless an analysis
assumption or robustness setting is being varied.
